## 1. 产品概述

CineCraft（视听创客）是一款面向自媒体人、导演和影视专业学生的"一站式运镜与分镜策划工作站"。通过 AI 辅助，将模糊的画面创意转化为极其专业的摄影机运动指令和视觉分镜，并提供一个专业、沉浸的中后台环境来管理这些创意资产。

- **核心价值**：降低专业影视制作门槛，让创意人员快速生成好莱坞级别的镜头指令
- **目标用户**：自媒体创作者、独立导演、影视专业学生、视频策划人员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深邃黑 (#0D0D0D) + 科技蓝 (#00D4FF) + 琥珀金 (#FFB800)
- **辅助色**：中性灰 (#1A1A1A, #2D2D2D, #404040)
- **按钮风格**：圆角微凸，渐变背景，悬停发光效果
- **字体选择**：
  - 标题：Orbitron（科技感显示字体）
  - 正文：Inter（清晰易读）
  - 代码/提示词：JetBrains Mono（等宽字体）
- **布局风格**：左侧固定导航 + 右侧内容区，卡片式项目展示
- **图标风格**：线性图标，2px 描边，圆角端点

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 登录页 | 左侧视觉区 | 全高背景图，渐变遮罩，品牌 Logo + 标语，科技感动画粒子 |
| 登录页 | 右侧表单区 | 玻璃态卡片，输入框带图标，渐变按钮，链接悬停效果 |
| 工作台大盘 | 顶部统计栏 | 四宫格统计卡片，数字动画，图标装饰 |
| 工作台大盘 | 项目列表 | 网格布局，悬停缩放，缩略图预览，操作按钮浮现 |
| 工作台大盘 | 新建项目抽屉 | 右侧滑入，表单验证，拖拽上传封面 |
| 分镜工作区 | 参数面板 | 分组表单，下拉选择器，标签输入，滑块控件 |
| 分镜工作区 | 结果展示区 | 代码高亮显示，复制按钮，图片预览网格 |

### 4.3 响应式设计

- **桌面优先**：主要针对 1440px+ 宽度优化
- **平板适配**：侧边栏可折叠，卡片改为双列
- **移动端**：底部导航，单列布局，触摸优化

### 4.4 动效设计

- **页面切换**：淡入淡出 + 轻微位移
- **卡片悬停**：scale(1.02) + 阴影增强
- **按钮交互**：涟漪效果，加载状态旋转
- **数据加载**：骨架屏，渐进式显示
- **提示词生成**：打字机效果逐字显示

## 5. 专业运镜预设

### 5.1 运镜模式选项

| 模式名称 | 描述 |
|----------|------|
| 推拉摇移 | 经典摄影机运动，推近/拉远/摇摄/移动 |
| FPV 穿梭航拍 | 第一人称视角无人机穿梭镜头 |
| 大范围延时摄影 | 移动延时，展现时间流逝 |
| 环绕飞行 | 360度环绕主体拍摄 |
| 跟踪跟拍 | 跟随主体移动的镜头 |
| 升降镜头 | 垂直方向的升降运动 |

### 5.2 美学风格选项

| 风格名称 | 描述 |
